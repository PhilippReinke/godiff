# godiff

Simple tool that allows to compare two directories. So far only size comparison
is supported.

## Usage

```sh
godiff --help
```

## Installation

```sh
# option 1
git clone https://github.com/PhilippReinke/godiff.git
cd godiff
make install

# option 2
go install github.com/PhilippReinke/godiff@latest
```
